Setup notes (GitHub + Vercel)
  1. Keep it static: upload index.html, styles.css, and app.js to a GitHub repo.
  2. Publish your Google Sheet to the web: File → Share → Publish to web → pick the correct tab → publish.
  3. Paste the pubhtml URL (and gid if needed) above, then hit Reload.
  4. Deploy on Vercel: import the repo → Framework preset “Other” → deploy.

This dashboard reads only what your sheet publishes. If you unpublish the sheet, the dashboard can’t read it.
